¿Cómo funciona?

Esta función usa JSONBin.io (servicio gratuito) para guardar tu base de datos en internet. Una vez configurado, podrás abrir esta página desde cualquier computador y ver todas las órdenes de trabajo.

  1. Crea una cuenta gratuita en jsonbin.io
  2. En el dashboard, crea un nuevo Bin con este contenido: {"db":[],"contador":1}
  3. Copia el Bin ID (aparece en la URL como /b/xxxxxxxx)
  4. En tu perfil de JSONBin, crea una API Key (pestaña API Keys)
  5. Pega ambos datos abajo y haz clic en Guardar configuración
